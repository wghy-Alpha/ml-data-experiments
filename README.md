# ml-data-experiments
## 1. MovieLens 数据集 (GroupLens)
MovieLens 是推荐系统研究中最常用的基准数据集：

* **MovieLens 100K**: [点击下载 (ml-100k.zip)](https://files.grouplens.org/datasets/movielens/ml-100k.zip)
    * *说明：包含 1,000 名用户对 1,700 部电影的 10 万条评分数据。*
* **MovieLens 25M**: [点击下载 (ml-25m.zip)](https://files.grouplens.org/datasets/movielens/ml-25m.zip)
    * *说明：包含 6.2 万部电影的 2,500 万条评分数据。*
* **MovieLens 32M**: [点击下载 (ml-latest.zip)](https://files.grouplens.org/datasets/movielens/ml-latest.zip)
    * *说明：官方提供的最新完整版数据集（通常指 ml-latest）。*
